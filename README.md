# SIWA Api Gateway Apple Authorizer Demo Code

This repository contains the demo code for the `api-gateway-apple-authorizer` Serverless Application Repository (SAR) code.

There are two subdirectories in this repository:
- `api-gateway-cdk`: the CDK code for deploying the backend
- `siwa-demo`: the iOS code

Please see the READMEs in both directories for instructions on how to deploy them.
